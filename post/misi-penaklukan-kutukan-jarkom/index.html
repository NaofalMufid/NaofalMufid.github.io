Misi Penaklukan Kutukan Jarkom

Posted by: Thu Dec 27, 2018

Sebentar lagi ujian Jarkom lanjut tiba dan akan mendapa ujian mengalahkan Ksatria sihir bernama Mikrotik dari negeri Bata.Yang perlu di waspadai dari sang Ksatria Mikrotik adalah dungeon berbahaya yang akan dilewati.

  • Setting Router
  • Mikrotik konek LAN (DHCP Server)
  • Hotspot
  • Setting Login Page
  • Blokir Situs
  • Setting Waktu
  • Limit Bandwith

a. Setting Router

Sihir yang harus dikeluarkan dalam membuka dungeon pertama. Anggap si Mikrotik sudah terhubung ke Internet lewat WAN(port:1). Dan anda sebagai ksatria sihir dianggap kenal dengan WinBox.

  1. Koneksikan Mikrotik dengan AP(akses internet) dengan menghubungkanya dengan LAN diport WAN
  2. Koneksikan kabel LAN di port:2 ke Laptop/PC.
  3. Seting interface (biar mudah)
    • ether1 dinamai internet
    • ether2 dinamai lan
    • biarin yang wlan gitu aja atau dihidupin dmenu wireless
  4. Buka winbox seperti biasanya, masuk ke menu IP -> DHCP-Client -> + -> pilih ang ether1 -> sampai nanti ada tulisan bound di itunya.
  5. Isi yang pas, tes di CMD pake ping google.com misal.

b. Mikrotik konek LAN (DHCP Server)

Sihir untuk menyelesaikan dungeon ini lumayan sulit. Masih dengan alat sihir winbox

  1. Buka menu IP -> Address -> Masukan IP(ex:192.168.10.1⁄24 pilih interface ether2 LAN untuk antar pc dengan LAN, 192.168.5.1⁄24 *interface wlan*untuk pengguna hotspot)
  2. IP -> DHCP Server -> DHCP Setup -> pilih yang ether2
  3. IP -> Firewall -> NAT -> action:masquared -> disable -> enable LAN pc/laptop
  4. Ping google.com di cmd

c. Hotspot

Agar sihir kita terpancar di dungeon menengah. Sihir kelas rendah bisa saja menembus dinding pengahalang.

  1. IP Address tadi disetting di step b nomor 1
  2. Wireless -> klik centang agar wlan aktif -> double klik wlan -> station:AP Bridge, ssid:namawifi
  3. IP -> Hotspot -> Hotspot Setup
  4. Seting user : IP -> Hotspot -> user -> + (untuk nambah user misal) dan IP -> Hotspot -> user profiles(pengaturan user sperti limit kecepatan,user a bisa digunakan berapa)

d. Setting Login Page

Dungeon paling indah dalam misi kali ini. Mantra sihir yang digunakan “Files -> paste (dari themes yang anda punya).”

  1. IP -> Hotspot -> Server Profile -> pilih yang sedang aktif -> Tab menu General -> Html directory -> pilih yang theme tadi yang di pastekan di files

e. Blokir Situs

Pengguna sihir tingkat tinggi sangat dibutuhkan

  1. IP -> Firewall -> NAT ->

  2. action : drop, chain : forward, address list : IP addressnya berapa, content : situs yang diblok.

    Itu kadang nggak jadi ini aja

    • New terminal ketik : ip firewall filter tekan enter
    • ketik : add action=drop chain=forward content=namasitus.com src-address=192.168.5.1⁄24
    • keterangana content=gugel.com(di isi nama situs yang akan diblokir), src-address=ip yang akan terdampak blokir, misal kasus ini tadi setting ip di step b nomor 1 ip hotspot kan 192.168.5.1⁄24 berarti nanti pengguna hotspot tidak bisa akses gugel.com. kalau lan berarti yang masukin aja ip lan.

hati-hati banyak pengguna sihir tingkat tinggi terjebak di dungeon ini.

f. Setting Waktu

  1. System -> NTP Client -> enable -> primary NTP(copy dari ntp client mikrotik indonesia di gugel)
  2. System -> clock atur sesuai waktu kita
  3. IP -> Firewall -> + new firewall rule -> tambahan di extra, timenya diatur mau jam berapa aja.

dan ksatria sihir lengah lagi dalam dungeon gelap ini.

g. Limit Bandwith

Dungeon bercahaya yang menuntuk ke jalan tikungan yang lurus.

  1. Hotspot -> user profile -> cari aja rate limit ok dokey kalau nyasar bilang dong.

Misi dalam penaklukan Kutukan Jarkom masih panjang akan tetapi beberapa ksatria sihir gugur dan kembali ke markas untuk menyiapkan kstaria sihir hebat lainya. Selamat belajar dan jangan makan sembarangan
